학습한 태그 1, 2일차 : <h1></h1> : 구획 제목 <strong></strong> : 강조 <br> : 개행 <p></p> : 문단 생성 3일차 : <img src="asdf.jpg" width="100%"> : 이미지 삽입. <ul></ul> : 순서 없는 목록(unordered list). <ol></ol> : 순서 있는 목록(ordered list). 4일차 : <title></title> : 사이트 제목(디폴트 파일명). <meta charset="utf-8"> : 웹사이트의 본문을 저장하는방식. <body></body> : 본문을 묶는 태그. <head></head> : body 태그를 설명하는 태그. <html></html> : head, body 등의 태그를 감싸는 최고위 태그. <<a href="#"></a> : 하이퍼링크를 걸어주는 태그, href : 클릭 시 이동할 링크.
